Process

How TRINAY Works

A seamless pipeline from camera input to spoken guidance — in milliseconds.

📷

Capture

The device camera streams live frames at 30 fps. The processing pipeline selects every Nth frame for inference, ensuring smooth, lag-free operation.

🧠

Analyse

YOLOv8 identifies objects, faces, and hazards. Parallel modules run OCR, currency matching, and fall detection simultaneously.

⚡

Decide

The alert engine evaluates detected objects, computes distances in inches/metres, and decides whether a voice or vibration alert is required.

🗣️

Speak

Text-to-speech narrates the scene description, object identities, distances, and any safety warnings — giving the user a rich, eyes-free experience.
